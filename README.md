
# Potential Impact of Neurotoxicant Exposure During Fire Suppression on Firefighter Mental Health
## Description
This repository contains the R code (toxic metals.Rmd) for performing linear regressions, principal component regression (PCR) and Bayesian kernel machine regression (BKMR) to evaluate the associations between toxic metals and firefighters' mental health including depression, stress, anxiety, and post-traumatic stress disorder.
## How to Use
1. Make sure you have `R` installed, and install the required packages used in this project: `tidyverse`, `readxl`, `reshape2`, `pheatmap`, `Hmisc`, `knitr`, `writexl`, `bkmr`, `tibble`, `MASS`, `glmnet`, and `mice`.
2. Download the project files, including `toxic metals.Rmd` and the required Excel input files.
3. Open `toxic metals.Rmd` in `RStudio` or another `R` environment.
4. Update the file paths if needed so that all input datasets can be read correctly.
5. Run the `.Rmd` file chunk by chunk, or knit it to generate the output report.
